## Run Locally

> ‚ùóMake sure to install the [Backend](https://github.com/thesudeshdas/dashtra-backend) prior to the following steps.

1 - Clone the project

```bash
  git clone https://github.com/thesudeshdas/dashtra.git
```

2 - Go to the project directory

```bash
  cd dashtra
```

3 - Install dependencies

```bash
  npm install
```

4 - Add environment variables

```bash
  REACT_APP_SERVER_URL
```

5 - Start the server

```bash
  npm run start
```
